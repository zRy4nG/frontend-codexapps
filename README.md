# CodexAPPS - Discord Bots Management Platform

Plataforma de gerenciamento e vendas de bots para Discord desenvolvida com tecnologias modernas.

## Descrição

Website desenvolvido para a CodexAPPS, empresa especializada em criação e distribuição de bots para Discord. A plataforma oferece ferramentas completas para gerenciamento de clientes, controle de licenças, monitoramento de bots em tempo real e sistema de vendas integrado.


## Instalação

```bash
# Clone o repositório
git clone link_projeto

# Acesse o diretório
cd codexapps-platform

# Instale as dependências
npm install

# Configure as variáveis de ambiente
cp .env.example .env

# Inicie o servidor de desenvolvimento
npm run dev
